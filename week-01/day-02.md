One of the things i learnt about today was the **LISTING COMMAND** [ls].
- It's used to listing the content of a directory.
- To learn more details about a file, perform a long listing using the -l with the ls command [ls -l}.
  
  **SORTING**

  - The ls command is sorted alphabetically by file name but with the option [-t], it will sort the file by time stamp.
  - The [-s] will sort the files by size [ls -l -s].
  - The option [-r] will reverse the order of any type pf sort [ls -lSr].

    **THE ADMINISTRATIVE ACCESS**

    - Logging in as the root user provides administrative access, allowing for the execution of some of the priviledged commands.

   **THE SU COMMAND**;It allows you to temporarily act as a different user,It does that by creating a new shell.

  **THE SUDO COMMAND**;It allows a user to execute a command as another user without creating a new shell. It can also be used to switch to another account using  the [-u] option.

  **PERMISSIONS**; It determines the way different users can interact with a file or directory. It's named **chmod** instead of **chperm** because permissions used to be referred to as modes of access so **chmod** means change of mode of access.

  **CHANGING FILE OWNERSHIP**; The **chown** is used to change ownership of files and dierctories.
